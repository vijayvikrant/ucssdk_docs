ucssdk.utils.launchgui.start_ucs_kvm_session(handle, service_profile=None, blade=None, rack_unit=None, frame_title=None, need_url=False)[source]¶

Starts KVM session.

Launches the KVM session for the specific service profile, blade or rack_unit. - service_profile specifies an object of type lsServer.

Launches KVM session with which the service profile is associated.
  • blade specifies an object of type computeBlade. Launches KVM session of blade server.
  • rack_unit specifies an object of type computeRackUnit. Launches KVM session of rack Unit.
  • frame_title specifies the title of the frame window.
